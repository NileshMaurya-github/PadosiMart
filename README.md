## Getting Started

### Prerequisites

- Node.js 18+ & npm

### Installation

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to project directory
cd LocalMart

# Install dependencies
npm install

# Start development server
npm run dev
```

The app will be available at `http://localhost:8080`
